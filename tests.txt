The objective of the Mind Game is to allow the player to predict the color generated by the game. The player will be provided with the 10 distinct colors and 20 trails to predict the code. A player wins the game if he successfully predicts the code within 20 trials.

We have identified few critical stories to use them for a canary test . Below are as follows

pickFirstColor:
This test verifies if  the color generated by the player is same the color chosen by him.

generateColorTest:
This tests verifies if the game is generating the color code upon starting.


gameColorLengthTest:
This test verifies if the length of the color code generated by the game and user is equal to specified size in the user story.In this case six.
