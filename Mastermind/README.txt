1. The requirements for the game can be found in 'Mastermind Requirements' file available in repository
2. To run the program your system should be configured with
Java 8 or above and JUnit5 or above.
3.Test cases can be found under the test folder.
4.Main class can be found under the src folder.
5.Run the gradlew batch file to obtain the code coverage 
6.Code coverage can be found under the build folder.
